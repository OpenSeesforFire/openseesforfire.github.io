Section Commands for Thermo-Mechanical Analysis in OpenSees

This page introduces the commands of using various sections for thermo-mechanical analysis in OpenSees.

Thermo-mechaical Sections for beam-column elements

FiberSecThermal

This section is derived by modifying the existing fiber based section to include the interface for thermal action. The definition of FiberSecThermal for 2d and 3d model in OpenSees can be scripted as follows:

section fiberSecThermal $secTag { fiber $fiberLocy $fiberLocz $fiberArea $matTag; }

FiberGJThermal

FiberGJThermal is derived from "FiberSecGJ", which can be used for 3d beam-column elements.

section fiberSecThermal $secTag -GJ $GJ { fiber $fiberLocy $fiberLocz $fiberArea $matTag; }

Thermo-mechaical Sections for shell elements

ElasticIsotropicThermal

ElasticIsotropicThermal is created for analyses using shell elements in OpenSees, which could either be defined with constant thermal expansion ratio, or using thermal expansion ratios for concrete or Steel. For the latter, corresponding stiffness reduction will be also applied automatically.

nDMaterial ElasticIsotropic3DThermal $matTag $E0 $Poisson_ratio $Density $Thermal_expansion_ratio;

LayeredShellThermal

LayeredShellThermal is created for analyses using shell elements in OpenSees, which can be used for slabs of multiple layer. Each layer can be assigned with a thickness and a material type

section LayeredShellThermal $SecTag $NumLayers $layerThickness1 $matTag1 $layerThickness2 $matTag2....;

A sample is shown as below:

section LayeredShellThermal 1 13 4 0.01 4 0.01 4 0.005 3 0.000198 5 0.000198 4 0.004604 4 0.01 4 0.01 4 0.01 4 0.01 4 0.01 4 0.01 4 0.01 ;
